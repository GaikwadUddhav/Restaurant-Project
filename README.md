# Patil Table - Indian Fine Dining

This is a Next.js 15 application built in Firebase Studio, featuring a real-time table reservation system and an owner dashboard.

## How to push to your GitHub Repository

To push this code to `git@github.com:Abhi71-star/central3.0.git`, follow these steps on your local machine:

1. **Download the project files** from Firebase Studio.
2. **Open your terminal** in the root directory of the project.
3. **Initialize and push** using these commands:

```bash
# Initialize git
git init

# Add the remote repository
git remote add origin git@github.com:Abhi71-star/central3.0.git

# Stage all files
git add .

# Commit changes
git commit -m "Initial commit of Patil Table app from Firebase Studio"

# Push to the main branch
git branch -M main
git push -u origin main
```

## Features
- **Real-time Reservations:** Book one of 15 tables with instant availability checks.
- **Owner Dashboard:** Manage orders, bookings, and physical tables.
- **Indian Cuisine Menu:** Fully localized with INR (₹) pricing.
- **AI Menu Designer:** Generate creative dish descriptions using Genkit.
- **Firebase Integration:** Powered by Firestore and Firebase Authentication.
